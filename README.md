# 📧 Spam Email Classification Project

## 🔍 Overview

This project implements a spam email detection system using feature extraction based on the UCI Spambase dataset.

## ⚙️ Key Features

* Extracts 57 features from raw email text
* Includes:

  * Word frequency patterns
  * Special character usage
  * Capital letter statistics

## 🧪 Implementation

* Custom feature extraction from `.txt` email files
* Converts emails into structured numerical vectors
* Tested on real spam and non-spam examples

## 📊 Example Results

* Non-spam emails show low keyword and symbol frequency
* Spam emails show higher usage of promotional words, links, and urgency indicators

## 🚀 Tech Stack

* Python
* NumPy
* Pandas
* Jupyter Notebook

## 📌 Project Goal

To understand how real-world spam filters represent emails numerically for machine learning models.
